About

Hi, My name is Siavash Fahimi and this is my first website. the process of building this website and learning many subjects for the first time was really rewarding and i am very excited that it’s almost completed.

Topic

At first when I decided to start the level 2 challenge for Holberton School, it seemed a pretty difficult challenge. The biggest question in my head was, "what should the website be about?" and to answer that question I googled “machine learning algorithm in javascript” why this? because Machine Learning is the subject that I’m passionate about, also the reason for applying at Holberton School. I chose javascript because I had to learn js for the website anyway, so it could save me time to just learn one language for both the website and the algorithm, beside HTML and CSS.

I landed on Burak Kanber's blog which he explained Genetic Algorithm in a short and simple way. But when I got to the code section of his blog, I felt totally lost and I couldn’t understand a line of that code. Then I decided to write the code for algorithm from scratch so I can really learn it.

I wish I had more time to write about the whole learning process and all the interesting challenges that I faced to write the algorithm and build the website, but I must submit my project in about three hours and I’m kind of behind for my content. But I will deffinatelly keep wrking on this page ang I'll host it on GitHub projec page.
